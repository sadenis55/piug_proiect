About the Bootcamp

Launch your web development career in this immersive bootcamp! Learn the essentials: HTML, CSS, and JavaScript. Then dive deep into React, a popular library for building dynamic web interfaces. By the end, you'll build modern web applications from scratch. Land your dream job in web development!

The Schedule

This is the schedule of the 2-week Programming Bootcamp. Be aware because changes may happen.

Week 1

  • Monday-Tuesday:
    • Mornings: Intro to Web Development
    • Afternoons: Hands-on Projects
    • Evenings: Group Activities & Workshops (Optional: Guided Hike)
  • Wednesday: Day Trip
  • Thursday-Friday:
    • Mornings: Introduction to JavaScript Libraries
    • Afternoons: Building with React
    • Evenings: Campfire & Code Review

Week 2

  • Monday-Tuesday:
    • Mornings: Advanced React Techniques
    • Afternoons: Project Focus
    • Evenings: Portfolio Development
  • Wednesday: Mock Interviews & Career Coaching
  • Thursday-Friday:
    • Project Presentations
    • Bootcamp Graduation
    • (Optional: Farewell Hike)
